Task 4

In this task the main quest was to make two function that implement the boolean operator XOR. The two makes med same output, but differen input. First one: a ⊕ b = ( a ∨ b ) ∧ ¬ ( a ∧ b ), and second one: c ⊕ d = (c ∧ ¬ d) ∨ (¬ c ∧ d ). The result can be seen in the console log.
