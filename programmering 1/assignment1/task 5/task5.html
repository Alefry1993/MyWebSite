Modulo

Modulo is an operator used for many different tasks to solve problems more easy. Modulo is first of all know for returning you the rest of an divided mathematical equation. Like if you have 9/2 is 4,5 and that is not an integer. If you use modulo in javascript it will return 1 like it is calculating 8/2 = 4 (+1) in rest, because four is integer.

To type modulo in javascript you type 9%2 and javascript will do the rest and return 1. This will give you the knowing if the number is odd or even. That knowing can help you in many other different tasks. Modulo is actually used to calculate the controlldigets ( The last diget) in an IBAN back account number and birth number.
